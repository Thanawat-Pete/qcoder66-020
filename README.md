# Qcoder

## คำอธิบาย

โปรเจกต์นี้เป็นที่เก็บไฟล์ข้อความธรรมดาที่มีคำทักทายในหลายภาษา (ไทยและอังกฤษ)

## เนื้อหาในโปรเจกต์

- `Hello.txt` - ไฟล์คำทักทาย ประกอบด้วย "สวัสดี!", "Hello!", "Greeting!"
- `World.txt` - ไฟล์คำทักทาย ประกอบด้วย "ชาวโลก!", "World!", "Human!"
- `AGENTS.md` - เอกสารสำหรับ Qoder เกี่ยวกับโครงสร้างของโปรเจกต์

## ประวัติการพัฒนา

โปรเจกต์นี้เริ่มต้นด้วยการสร้างไฟล์ข้อความสองไฟล์และมีการแก้ไขเล็กน้อย

## หมายเหตุ

โปรเจกต์นี้ไม่มีระบบ build, test หรือ lint commands เนื่องจากเป็นเพียงไฟล์ข้อความธรรมดา
